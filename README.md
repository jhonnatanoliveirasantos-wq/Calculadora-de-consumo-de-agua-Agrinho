# Calculadora de Desperdício de Água 🌾💧
Este projeto foi desenvolvido para o Concurso Agrinho 2026 (Subcategoria Programação).
## 📋 Sobre o Projeto
O objetivo é ajudar o pequeno produtor rural com 0 cálculo da quantidade de água que pode economizar, mudando o tradicional sistema de irrigaçã para o gotejamento, aliando tecnologia com preservação ambiental.
## ♿ Recursos de Acessibilidade
- **Alto Contraste:** Facilita a leitura para pessoas com baixa visão.
- **Aumento de Fonte:** Permite aumentar o tamanho do texto dinamicamente.
- **HTML Semântico:** Facilita o uso de leitores de ecrã.
## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (JS)
